使い方

  1. Download ボタンで PNG をダウンロード → Word / PowerPoint / Keynote に貼り付け
  2. R Code ボタンでコードを表示 → Copy して RStudio に貼り付け
  3. コード内の limits, breaks, name を編集して軸をカスタマイズ
  4. 自分のデータを geom_point() / geom_line() で追加してプロット
  5. 印刷して手書きでデータを書き込む使い方もできます

MedGraph Free について

MedGraph Free は、医学部生がレポートや発表で使えるグラフテンプレートを無料で提供するサイトです。 生化学（Michaelis-Menten プロット、Lineweaver-Burk プロット、pH 滴定曲線）、 生理学（酸素解離曲線、Frank-Starling 曲線、活動電位、心臓 PV ループ、フローボリューム曲線）、 薬理学（用量反応曲線、薬物動態曲線、治療域テンプレート）、 微生物学（細菌増殖曲線、MIC 測定、Time-Kill 曲線、ディスク拡散法）、 免疫学（一次/二次免疫応答、ELISA 標準曲線、フローサイトメトリー）、 疫学（流行曲線、2×2 分割表、PPV/NPV）、 臨床研究・統計（ROC 曲線、Kaplan-Meier 生存曲線、フォレストプロット、Bland-Altman プロット） など、医学部の全23分野をカバーする54種類のテンプレートを揃えています。

すべてのテンプレートはデータが入っていない白紙の状態で、自分の実験データや臨床データをプロットして使うことができます。 各テンプレートには R（ggplot2）のソースコードが付属しており、コピー＆ペーストですぐにカスタマイズ可能です。 軸のスケール変更、データの追加、色の変更など、自由に編集できます。 また、PNG 画像としてダウンロードして PowerPoint や Word に貼り付け、印刷して手書きで記入する使い方もできます。

全コンテンツは CC0（パブリックドメイン）ライセンスで公開しており、 商用利用・非商用利用を問わず、クレジット表記不要で自由にご利用いただけます。 医学部のレポート、卒業論文、学会発表のスライド、研究論文の図表作成などにご活用ください。
